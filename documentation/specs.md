# Document de specification

## 1 - Projet

SPEC 1.1 : Le projet unity doit être structuré proprement et avoir des dossiers pour le code source, les scènes, les assets, materiaux, etc ... 

## 2 - Capacité technique

SPEC 2.1 : Le jeu doit être rejoignable en CO-OP (au moins 4 joueurs)

SPEC 2.2 : Le joueur doit pouvoir se déplacer dans les 4 directions (haut, bas, droite, gauche) et dans les combinaisons de ces directions.

SPEC 2.3 : Le joueur doit pouvoir interagir avec des éléments de la carte (ex: porte, coffre, etc...). Le système d'interaction doit être générique pour permettre d'ajouter facilement de nouveau type d'interaction.

SPEC 2.4 : Le joueur doit pouvoir utiliser un objet principal et changer d'objet principal
